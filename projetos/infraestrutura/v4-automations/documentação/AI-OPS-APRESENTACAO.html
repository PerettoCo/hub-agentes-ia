O Diagnóstico

O CAOS QUE NÃO VOLTA

Antes da AI Ops, a operação dependia de coleta manual, planilhas dispersas e decisão baseada em achismo. O custo de escala era proibitivo.

📉

Antes (Sem AI Ops)

  • Coleta manual de dados de anúncios (2h/account/semana)
  • Relatórios em planilhas descentralizadas
  • Decisão baseada em feeling, não em dados
  • Análise de 1 plataforma por vez
  • Escala limitada a 3-5 clientes por account
  • Erro de coleta: ~15%
📈

Depois (Com AI Ops)

  • Coleta automática via API (5 min/account)
  • Relatórios centralizados com template V4
  • Decisão baseada em dados reais + análise IA
  • 3+ plataformas simultâneas (Google, Meta, Bing)
  • Escala para 15-20 clientes por account
  • Erro de coleta: <1%

Setup

1 COMANDO. TUDO PRONTO.

De zero a relatório rodando em menos de 10 minutos.

Instalação

# Único comando — faz tudo:
python3 v4-automations/setup/install_all.py

O script guia por 9 passos: pré-requisitos → venv → dependências → credenciais interativas → clientes → OAuth → teste de email → relatório sample → crons.

Comandos do dia a dia

# Relatório ilustrativo (sem API)
python3 scripts/gt/relatorio_trafego.py \
  --cliente "Cliente A" --days 7 --sample

# Relatório real por email
python3 scripts/gt/relatorio_trafego.py \
  --cliente "Cliente X" --days 7 \
  --format html --email gt@v4company.com

# Todos os clientes de uma vez
python3 scripts/gt/relatorio_trafego.py \
  --all --days 7 --format html \
  --email gt@v4company.com

O que instala em 1 comando

  • Virtual environment (.venv)
  • 10 dependências Python
  • .env com credenciais (interativo)
  • clientes.json (cadastro guiado)
  • Google OAuth (token.json)
  • Teste de pipeline de email
  • Relatório sample gerado
  • Crons automáticos instalados
