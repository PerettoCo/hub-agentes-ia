## 🧠 Como Funciona

1. Acesse o portal, faça login com seu usuário e senha
2. Você é redirecionado automaticamente para **seu subdomínio exclusivo**
3. Lá dentro, você conversa com o agente de IA — ele **lembra de tudo** entre sessões
4. Precisa de ajuda especializada? Chame `@especialista` na conversa

## 🚀 Primeiros Passos

1. Faça login em [ia.fvmarketing.com.br](https://ia.fvmarketing.com.br)
2. Explore seu subdomínio — converse com o agente
3. Teste um `@especialista` — por exemplo: `@copy-content, escreva um texto sobre [assunto]`
4. Navegue pelo catálogo de skills em [skills.fvmarketing.com.br](https://skills.fvmarketing.com.br)
5. Se tiver dúvidas, é só perguntar pro agente — ele conhece o ecossistema todo
